# 🤖 AI Agents Basics

## 📌 Description

This repository contains the basic implementation of AI Agents using Python.
It demonstrates how an intelligent agent can perceive its environment, make decisions, and take actions to achieve a goal.

---

## 🧠 What is an AI Agent?

An AI Agent is a system that:

* **Perceives** its environment (input)
* **Processes/Thinks** using logic or models
* **Acts** to achieve a goal

👉 Simple Formula:
Agent = Perception + Decision + Action

---

## 🧩 Types of AI Agents

1. Simple Reflex Agent
2. Model-Based Agent
3. Goal-Based Agent
4. Utility-Based Agent
5. Learning Agent

---

## ⚙️ How It Works

1. Take input from user/environment
2. Apply decision logic
3. Return action/output
